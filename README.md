# capslox-by-karabiner

使用karabiner实现capslox的大部分capslock快捷键功能

参考项目[capslox-karabiner](https://github.com/yqchilde/capslox-karabiner.git)

## 使用方法

1. 下载 [Karabiner-Elements](https://karabiner-elements.pqrs.org/) 并安装好
2. 在默认浏览中输入下面网址，并回车

   ```
   karabiner://karabiner/assets/complex_modifications/import?url=https://raw.githubusercontent.com/wzk-chi/capslox-by-karabiner/main/capslox-by-karabiner.json
   ```
3. 跳转至Karabiner-Elements并导入
4. 选择enable ALL


## 主要修改

优化capslock的逻辑，不再使用cmd+shift+option作为capslock的功能，防止快捷键冲突

添加了 capslock+\` 打开浏览器
